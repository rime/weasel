安裝輸入法

本品適用於 Windows XP,  32/64位 Windows 7

欲將【小狼毫】註冊到繁體中文（臺灣）鍵盤佈局，
請通過命令行介面啓動安裝程序，並指定選項 /T ：

> weasel-x.x.x-installer.exe /T

----------------------------------------------------------
使用輸入法

選取輸入法通知區的【中】字樣圖標，開始用小狼毫寫字。
通過快捷鍵 Ctrl+` 或 F4 呼出方案選單、切換輸入方式。

----------------------------------------------------------
定製輸入法

右擊托盤圖標，快速訪問配置管理工具。
修改配置文件，於「重新載入」後生效；
修改詞庫，通過「部署方案」生效。
用戶配置文件位於 %AppData%\Rime 調教Rime輸入法你會常用到。
爲免自定義內容會在輸入法升級後丟失，請勿修改安裝目錄中的
文件；如需定製預設輸入方案，應在用戶配置目錄重命名該方案。

更新日誌

小狼毫 0.9.6
  * 關機時妥善保存數據，降低用戶詞庫損壞機率；執行定期備份
  * 新增基於【朙月拼音】的衍生方案：
    * 【語句流】，整句輸入，空格分詞，回車上屏
    * 【雙拼】，兼容自然碼雙拼方案，演示拼寫運算常用技巧
  * 修復BUG：簡拼「z h, c h, s h」的詞候選先於單字簡拼
  * 修復BUG：「拼寫運算」無法替換爲空串
  * 完善拼寫運算的錯誤日誌；清理調試日誌

小狼毫 0.9.5
  * Rime 獨門絕活之「拼寫運算」
  * 升級【朙月拼音】，支持簡拼、糾錯；增設【簡化字】方案
  * 升級【倉頡五代】，以倉頡字母顯示編碼
  * 重修配色方案【碧水／Aqua】、【青天／Azure】

小狼毫 0.9.4
  * 增設編碼反查功能，預設方案以「`」爲反查的引導鍵
  * 修復Windows XP中西文狀態變更時的通知氣球

小狼毫 0.9.3
  * 新增預設輸入方案【五笔86】、【臺灣正體】拼音
